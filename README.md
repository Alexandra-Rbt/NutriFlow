**NutriFlow** is a comprehensive web application designed to help users monitor their daily caloric intake efficiently. Built with a focus on clean code and database integrity, this project demonstrates full-stack development capabilities using Python and the Django framework.

---

##  1. Key Features
* **User Authentication:** Secure sign-up and login functionality for personalized data tracking.
* **Calorie Journal:** Full CRUD (Create, Read, Update, Delete) capabilities for adding and managing daily food entries.
* **Automatic Calculations:** Real-time summarization of calories based on food portions and nutritional data.
* **Responsive Dashboard:** An intuitive UI/UX for visualizing daily progress across different devices.
* **Robust Database:** Seamless integration with MySQL for reliable data persistence.

##  2. Tech Stack
* **Backend:** Python 3.x, Django Framework
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript
* **Version Control:** Git & GitHub
* **AI Tools:** Used AI assistance for debugging, query optimization, and generating unit tests.

## 3. Installation & Setup
To run this project locally, follow these steps:

1. **Clone the repository:**

```bash
git clone [https://github.com/Alexandra-Rbt/NutriFlow.git](https://github.com/Alexandra-Rbt/NutriFlow.git)
```

2. **Install dependencies:**
(Ensure you have a virtual environment active)

```bash
pip install -r requirements.txt
```
5. **Database Configuration:** Update the DATABASES section in nutriflow/settings.py with your local MySQL credentials.
6. **Apply Migrations:**

```bash
python manage.py migrate
```
8. **Start the server:**
```bash
python manage.py runserver
```
